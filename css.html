CSS Nedir?

Yayın Tarihi: 6 Mayıs 2025

  • CSS (Cascading Style Sheets), web sayfalarına görsel biçim kazandırmak için kullanılan bir stil dilidir. Renk, yazı tipi,boşluk, hizalama gibi görsel detaylar CSS ile düzenlenir.



    CSS ne işe yarar?

  • CSS, HTML ile oluşturulan sayfa yapısına estetik katmak için kullanılır. Yazı rengini değiştirebilir, kutular oluşturabilir, arka planlar ekleyebilir, öğeleri konumlandırabiliriz.



    CSS nasıl kullanılır?

  • Satır içi (inline): HTML etiketinin içine direkt yazılır.

  • Dahili (internal):style etiketi içinde yazılır.

  • Harici (external): Ayrı bir "css" dosyasına yazılır ve sayfaya "link" ile bağlanır.



    CSS dosyası nedir?

  • CSS kodlarını HTML’den ayrı tutmak için .css uzantılı dosyalar kullanılır. Bu sayede tüm stil ayarları tek bir yerden kontrol edilebilir.



    CSS ile neler yapılabilir?

  • Yazı tipi, rengi, boyutu ayarlanabilir

  • Arka plan ve kenarlık verilebilir

  • Sayfa bölümleri hizalanabilir

  • Responsive (mobil uyumlu) tasarımlar yapılabilir

  • Hover gibi etkileşimler eklenebilir



    CSS olmadan site nasıl görünür?

  • CSS kullanılmazsa site sadece düz yazı ve yapısal HTML öğeleri ile görünür. Estetikten uzak, sade ve karışıktır.
